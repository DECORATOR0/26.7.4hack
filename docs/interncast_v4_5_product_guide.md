## 产品概述

InternCast 是一个面向足球比赛视频的自动解说与事项回看系统。它的输入是一场完整比赛视频，输出是可交互的比赛回看页面，以及可用于配音、字幕、集锦讲解和多语言展示的解说材料。

当前 V4.5 版本以“德国 vs 库拉索”比赛为样例，自动识别并整理了 43 个关键事项，其中包括 8 个由记分牌 OCR 比分跳变确认的进球。系统把这些事项转成可点击的前端按钮、对应的视频片段和解说字幕，让用户可以按事件类型快速回看比赛。

## 前端功能

前端演示页支持以下功能：

- **事项回看**：按进球、射门机会、角球、任意球、判罚争议、换人等类型浏览关键事项。
- **片段播放**：每个事项都有对应的 10 秒左右回看片段。
- **一分钟视频 Demo**：提供一个整合后的短视频演示版本。
- **多语言文案**：支持中文、英文、西班牙语、法语。
- **中文风格切换**：支持激情风格和稳健风格两种表达。
- **手机访问**：支持通过公开链接或二维码在手机浏览器打开。

当前前端事项统计：

| 事项类型 | 数量 | 说明 |
|---|---:|---|
| 进球 | 8 | 仅使用记分牌 OCR 比分跳变确认 |
| 射门机会 | 12 | 包括明确射门、扑救和门前威胁 |
| 角球 | 6 | 来自角球区域和禁区站位画面 |
| 任意球 | 6 | 包括前场定位球和任意球攻防 |
| 判罚争议 | 3 | 包括裁判沟通、身体接触和争议场面 |
| 换人 | 8 | 包括场边人员调整和换人牌画面 |

V4.5 外部展示端不保留点球、越位、半场/全场、庆祝等事项类型，以降低误触发和重复展示。

## 输入与输出

### 输入

系统主要输入为：

| 输入 | 作用 |
|---|---|
| 完整比赛视频 | 提供全场画面、比分牌、比赛时间和比赛过程 |
| 比赛基础信息 | 提供对阵双方、赛事背景和输出约束 |
| 视频帧分析结果 | 为射门、角球、任意球、换人、判罚争议等事项提供视觉证据 |
| 记分牌 OCR 结果 | 用于确认比分跳变和进球时间 |

### 输出

系统输出包括：

| 输出 | 说明 |
|---|---|
| 前端演示页面 | 可点击回看事项、播放视频片段、切换语言和风格 |
| 秒级事项记录 | 用于前端展示的结构化事项，比赛时间精确到 10 秒粒度 |
| 解说脚本 | 面向配音和展示的中文解说稿，包含激情和稳健两种口吻 |
| 多语言脚本 | 中文、英文、西班牙语、法语版本的解说文本 |
| 字幕脚本 | 按视频时间戳排列的字幕内容 |
| 集锦讲解脚本 | 可用于短视频或 Demo 讲解的分段文案 |
| 技术统计摘要 | 基于当前保留事项得到的统计结果 |

## 生成流程

V4.5 的自动生成流程如下：

```text
完整比赛视频
  -> 抽帧与视频时间索引
  -> 多模态模型生成画面描述
  -> 事件 Agent 提取关键事项
  -> Guardrail 过滤不可靠事项
  -> 记分牌 OCR 校准进球
  -> 生成秒级事项记录
  -> 生成解说、字幕、多语言脚本
  -> 生成前端事项数据和回看片段
  -> 输出可访问的前端演示页
```

V4.5 的关键规则：

- 进球只采纳记分牌 OCR 的比分跳变。
- 前端事项只保留进球、射门机会、角球、任意球、判罚争议、换人。
- 前端事项时间精确到 10 秒粒度，便于生成可点击回看。
- 解说和脚本按分钟级叙述，保证阅读和配音更自然。
- 不编造缺少证据支持的球员姓名、号码、助攻、VAR 或红黄牌。
